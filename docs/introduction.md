# 项目概述

EZ Admin NaiveUI 是一个基于 Vue 3 + Naive UI + TypeScript 的企业级后台管理系统，提供了完整的用户权限管理系统和常用的业务模块。

## 项目特性

- **现代化技术栈**: 采用 Vue 3、TypeScript、Vite 等现代前端技术
- **组件化架构**: 基于组件化的开发模式，提供可复用的 UI 组件
- **类型安全**: 完整的 TypeScript 类型定义，提供良好的开发体验
- **权限管理**: 内置完整的用户、角色、菜单权限管理系统
- **响应式设计**: 支持响应式布局，适配不同设备
- **国际化支持**: 内置中文语言包，支持扩展多语言
- **主题切换**: 支持亮色/暗色主题切换
- **开发友好**: 提供丰富的开发工具和配置化能力

## 核心功能

### 用户管理
- 用户信息的增删改查
- 用户状态管理（启用/禁用）
- 用户角色分配
- 用户部门管理

### 角色管理
- 角色信息的增删改查
- 角色权限配置
- 角色用户分配

### 菜单管理
- 菜单结构的动态配置
- 菜单权限控制
- 支持多级菜单

### 部门管理
- 部门层级结构管理
- 部门用户分配

### 字典管理
- 系统字典数据的统一管理
- 支持字典项的动态配置

### 日志管理
- 操作日志记录和查询
- 系统异常日志管理

### 文件管理
- 文件上传下载功能
- 文件存储管理
