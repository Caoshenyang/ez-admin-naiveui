# 部门管理

## 功能规划

- ✅ 部门层级结构管理
- ✅ 部门信息的增删改查
- ✅ 部门用户分配
- ✅ 部门状态管理

## 预期字段

### 部门字段

| 字段名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| 部门名称 | string | ✅ | 部门显示名称 |
| 部门编码 | string | ✅ | 部门唯一标识 |
| 父级部门 | number | ❌ | 父级部门ID |
| 负责人 | string | ❌ | 部门负责人 |
| 联系电话 | string | ❌ | 部门联系电话 |
| 排序号 | number | ❌ | 显示排序 |
| 状态 | number | ✅ | 部门状态 |
