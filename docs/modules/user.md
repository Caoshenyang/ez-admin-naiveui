# 用户管理

用户管理模块提供了完整的用户生命周期管理功能，包括用户的创建、查询、更新、删除以及状态管理。

## 功能特性

- ✅ 用户信息的增删改查
- ✅ 用户状态管理（启用/禁用）
- ✅ 用户搜索和筛选
- ✅ 批量删除用户
- ✅ 用户详情查看
- ✅ 表单验证和错误提示

## 页面结构

用户管理页面由以下组件组成：

1. **搜索表单**: 支持用户名、昵称、邮箱关键词搜索
2. **操作按钮组**: 新增、批量删除、刷新等操作按钮
3. **用户列表表格**: 显示用户信息，支持分页、排序
4. **用户表单**: 新增/编辑用户的表单弹窗
5. **用户详情**: 查看用户详细信息

## 操作权限

| 操作 | 权限标识 | 说明 |
|------|----------|------|
| 查看用户 | `sys:user:view` | 查看用户列表和详情 |
| 新增用户 | `sys:user:add` | 创建新用户 |
| 编辑用户 | `sys:user:edit` | 修改用户信息 |
| 删除用户 | `sys:user:delete` | 删除单个用户 |
| 批量删除 | `sys:user:delete` | 批量删除用户 |

## 使用说明

### 查看用户列表

1. 进入用户管理页面
2. 系统自动加载用户列表
3. 支持分页浏览，默认每页10条记录

### 搜索用户

1. 在搜索框输入关键词（用户名、昵称或邮箱）
2. 点击搜索按钮或按回车键
3. 系统筛选匹配的用户
4. 点击重置按钮清除搜索条件

### 新增用户

1. 点击"新增"按钮
2. 填写用户表单信息
3. 点击"确定"提交表单
4. 系统验证表单数据并保存

### 编辑用户

1. 在用户列表中点击"编辑"按钮
2. 修改用户信息（用户名不可编辑）
3. 点击"确定"保存修改

### 删除用户

**单个删除：**
1. 在用户列表中点击"删除"按钮
2. 确认删除操作

**批量删除：**
1. 勾选需要删除的用户
2. 点击"批量删除"按钮
3. 确认批量删除操作

### 查看用户详情

1. 在用户列表中点击"查看"按钮
2. 弹出用户详情模态框
3. 显示用户的完整信息

## 表单验证规则

- **用户名**: 必填，3-20个字符，只能包含字母、数字和下划线
- **密码**: 新增时必填，6-20个字符
- **昵称**: 必填，2-20个字符
- **邮箱**: 选填，必须是有效的邮箱格式
- **手机号**: 选填，必须是有效的11位手机号码

## 数据状态

### 用户状态

- **启用 (1)**: 用户正常使用系统
- **禁用 (0)**: 用户被禁用，无法登录系统

### 性别选项

- **男 (1)**
- **女 (2)**

## 注意事项

1. 用户名在系统中唯一，新建时会验证重复性
2. 删除用户时会检查用户是否有未完成的任务或关联数据
3. 禁用用户后，该用户将无法登录系统，但数据保留
4. 邮箱和手机号为选填项，但建议填写以便找回密码等功能
5. 密码字段在编辑时不显示，可留空表示不修改密码
