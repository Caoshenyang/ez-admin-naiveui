# 菜单管理

::: warning 注意
菜单管理模块正在开发中，目前仅提供基础页面框架。完整功能将在后续版本中提供。
:::

## 功能规划

- ✅ 菜单的增删改查
- ✅ 菜单层级结构管理
- ✅ 菜单权限配置
- ✅ 菜单图标设置

## 预期字段

### 菜单字段

| 字段名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| 菜单名称 | string | ✅ | 菜单显示名称 |
| 菜单类型 | number | ✅ | 菜单类型（目录/菜单/按钮） |
| 父级菜单 | number | ❌ | 父级菜单ID |
| 菜单路径 | string | ❌ | 路由路径 |
| 菜单图标 | string | ❌ | 菜单图标 |
| 排序号 | number | ❌ | 显示排序 |
| 权限标识 | string | ❌ | 权限标识 |
| 状态 | number | ✅ | 菜单状态 |

### 菜单类型

- **目录 (1)**: 菜单目录，可包含子菜单
- **菜单 (2)**: 具体页面菜单
- **按钮 (3)**: 页面内按钮权限

## 预期功能

### 菜单树结构

- 支持无限级菜单嵌套
- 拖拽排序功能
- 菜单展开/折叠

### 菜单权限

- 菜单访问权限
- 按钮操作权限
- 路由权限控制
