# 角色管理

::: warning 注意
角色管理模块正在开发中，目前仅提供基础页面框架。完整功能将在后续版本中提供。

## 功能规划

- ✅ 角色的增删改查
- ✅ 角色权限分配
- ✅ 角色用户关联
- ✅ 角色状态管理

## 预期字段

### 角色列表字段

| 字段名 | 类型 | 说明 |
|--------|------|------|
| 角色名称 | string | 角色显示名称 |
| 角色编码 | string | 角色唯一标识 |
| 状态 | number | 角色状态 |
| 创建时间 | string | 角色创建时间 |

### 角色表单字段

| 字段名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| 角色名称 | string | ✅ | 角色显示名称 |
| 角色编码 | string | ✅ | 角色唯一标识 |
| 状态 | number | ✅ | 角色状态 |
| 权限配置 | array | ❌ | 角色拥有的权限列表 |
| 描述 | string | ❌ | 角色描述信息 |

## 预期功能

### 角色权限管理

角色可以分配以下权限：

- **菜单权限**: 控制用户可访问的菜单项
- **操作权限**: 控制用户的具体操作权限
- **数据权限**: 控制用户可访问的数据范围

### 角色用户关联

- 为用户分配角色
- 查看角色的用户列表
- 批量调整用户角色
